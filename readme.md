# Python和OpenCV仿Github默认头像



开发环境

* pycharm
* python 3.8
* opencv-python 4.5.1.48
* numpy 1.19.2

|      |        |                                                              |
| :--: | :----: | ------------------------------------------------------------ |
|  1   |  cv2   | 一个开源的跨平台计算机视觉库，图像处理，此处用来显示，保存图像 |
|  2   | numpy  | 开源的Python科学计算库，处理图像数组数据                     |
|  3   | random | 用于生成随机填充颜色与随机位置                               |
